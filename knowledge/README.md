# MeshBot Knowledge Base

This directory contains text files that serve as the knowledge base for MeshBot.

## Supported File Types

- `.txt` - Plain text files
- `.md` - Markdown files
- `.rst` - reStructuredText files
- `.py` - Python source files
- `.js` - JavaScript files
- `.html` - HTML files
- `.css` - CSS files

## Organization

You can organize files in subdirectories. The knowledge base will recursively scan all subdirectories.

## Example Content

Here are some example files to get started:

### meshcore_basics.txt
MeshCore is a decentralized communication network that allows devices to communicate directly without relying on centralized infrastructure.

Key features:
- Mesh networking topology
- End-to-end encryption
- No single point of failure
- Low power consumption
- Long range communication

### getting_started.md
# Getting Started with MeshBot

MeshBot is an AI agent that communicates through the MeshCore network.

## Basic Commands

- `ping` - Test connectivity
- `help` - Show available commands
- `search <query>` - Search knowledge base
- `contacts` - List available contacts

## Usage Tips

1. Keep messages concise for network efficiency
2. Use the knowledge base to store frequently needed information
3. The bot remembers conversation history for context

### troubleshooting.txt
Common MeshBot Issues and Solutions:

1. Bot not responding:
   - Check MeshCore connection
   - Verify power to MeshCore device
   - Check serial port configuration

2. Slow responses:
   - Network congestion
   - High message volume
   - AI model processing time

3. Memory issues:
   - Clear old conversation history
   - Check disk space for memory file
   - Restart bot if needed
